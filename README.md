# RHMediaPlayer-core

RHMediaPlayer核心Core库

### 引用方式：

在项目的Build.gradle里添加：
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
}
```
在项目的module里面添加：
```
dependencies {
	   implementation 'com.github.renhui:RHMediaPlayer-core:v0.1.2-beta'
}
```
